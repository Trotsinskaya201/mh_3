Краткое описание опыта работы с технологиями, использованными в проекте:

В рамках данного проекта был получен опыт работы с технологиями стека Spring Boot для создания веб-приложения на Java. Использовались Spring Security для настройки аутентификации и авторизации, Spring Data JPA для взаимодействия с базой данных PostgreSQL. Интерфейс пользователя реализован с использованием Thymeleaf и Bootstrap для адаптивного дизайна. Управление зависимостями проекта осуществлялось через Maven.
